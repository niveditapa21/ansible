************
To install packages at once:
____________________________
---
- name: installation of multiplepackages
  hosts: all
  tasks:
   - name: install two packages
     yum:
      name:
       - httpd
       - unzip
       - wget


Playbook2:
***********
Two tasks/plays at once:
---
- name: install httpd
  hosts: all
  tasks:
   - name: install httpd
     yum:
      name: httpd
      state: latest
   - name: restart web service
     service:
      name: httpd
      state: restarted
